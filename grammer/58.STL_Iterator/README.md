### 반복자(Iterator)

- 컨테이너의 요소에 대해 "추상적"인 접근을 가능하게 하는 기능
  -> 추상적 : 컨테이너가 무슨 종류인지, 어떻게 구현되어 있는지 알 필요 없다.
  //vector, array, map, set 다 first()라는 걸로 접근 가능하게 한다.
- 포인터와 유사하게 사용 가능
- 대부분 컨테이너는 반복자를 사용해 순환 가능
  -> stack, queue는 예외

### 반복자의 선언

- 반복자가 사용될 컨테이너의 타입을 명시적으로 표기

### Auto 키워드

- 자동으로 타입을 추정해주는 키워드
- 타입이 길어질 경우 사용하면 편리
  -> 남용할 경우 코드의 가독성을 해칠 수 있다.
