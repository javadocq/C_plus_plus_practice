### Local / Global Scope

- 지역 범위
  - 블록 {} 내의 범위
  - 함수의 매개변수는 함수 본체 내에서만 존재
  - 따라서 복사된 지역 변수들은 함수의 실행 중에만 존재

- static 지역 변수
  - static 한정어를 사용해 예외 변수 지정이 가능하다.
  - 초기화가 필요

- 전역 범위
  - 함수 밖에 정의된 변수는 어디서나 접근 가능
  - 전역 변수는 가급적 사용하지 않는 것이 좋다.
    -> 전역 상수는 OK

### 함수 호출의 동작 방식

    - Function call stack
    - LIFO
    - Stack Frame(Activation Record)
        -> 함수의 호출이 발생할 때마다 구분선이 생성된다.
        -> 함수의 지역 변수와 매개변수는 그 구분선 내에 생성된다.
        -> 함수의 호출이 끝나면 구분선 내의 메모리가 자동으로 해제된다.

    - 스택은 유한하고, stack overflow가 발생할 수 있다.

### 스택 메모리

- 코드 공간
- Static 변수 ,전역 변수, String 리터럴
- 스택(Stack) <- 빠르지만 작고 메모리가 자동으로 해제 된다.
- 힙 (Heap) <- 크지만 느리고 메모리를 직접 해제해주어야 한다. (동적할당)
