### 생성자

- 특수한 멤버 함수
- 객체가 생성될 떄 자동으로 호출된다.
- 초기화 목적으로 유용하게 사용된다.
- 클래스와 동일한 이름을 갖는 멤버 함수
- 반환형은 존재하지 않는다.
- 오버로딩 가능

### 소멸자

- 특수한 멤버 함수
- 객체가 소멸할 때 자동으로 호출된다.
- 메모리 및 기타 리소스(파일 close 등) 해제 목적으로 유용하게 사용된다.
- 클래스와 동일한 이름 앞에 "~"을 갖는 멤버 함수
- 반환형 및 파라미터는 존재하지 않는다.
- 오버로딩 불가능

### 기본 생성자

- 인자가 없는 생성자
- 클래스의 생성자를 직접 구현하지 않으면, 컴파일러가 기본적으로 만들어 준다.
- 인자가 없는 클래스 생성자도 구현을 해주는 게 좋다. -> 쓰레기값 항상 안전 방지
