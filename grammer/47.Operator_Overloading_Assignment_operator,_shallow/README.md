### 대입 연산자 오버로딩

- C++는 대입 연산자를 자동 생성해준다.
- 자동 생성된 대입 연산자는 얕은 복사를 수행한다. (얕은 복사란? 객체의 멤버 변수들을 단순히 복사하는 것)
- 따라서, 대입 연산자를 오버로딩하여 깊은 복사를 수행하도록 구현해야 한다. (깊은 복사란? 객체의 멤버 변수들이 가리키는 메모리 공간까지 복사하는 것)

- 기본 패턴
	```cpp
	Type & operator=(const Type& other) {
		if (this != &other) { // 자기 자신과의 대입 방지
			// 기존 자원 해제
			// 새로운 자원 할당 및 복사
		}
		return *this; // 대입 연산자 체이닝 지원
	}
	```
- 참조형으로 반환하여 추가적인 복사 연산을 하지 않도록 한다.
	-> 참조형으로 반환하지 않으면 어떤 일이 벌어질까?
		-> 대입 연산자 체이닝이 불가능해진다. (예: a = b = c;에서 b = c의 결과가 a에 대입되지 않음)
