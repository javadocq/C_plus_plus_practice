### Integer

- 정수를 표현 (4바이트)

### Floating point

- 실수를 표현
- float (4바이트)
- double (8바이트)

### Boolean

- true/false (1바이트)

### 상수

- 변수와 유사하지만, 초기화 이후 변할 수 없는 값
- 사용 목적 -> 실수 방지, 프로그램의 견고함

- 리터럴 상수 : 12, 3.14, "Hyungki" (r-values)
- 선언 상수 : const int favorite_number = 50;
- 상수 표현 : constexpr int age = 20;
- 열거형
- Defined : #define pi 3.1415926
