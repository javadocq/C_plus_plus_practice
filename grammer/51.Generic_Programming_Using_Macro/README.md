### 매크로를 사용한 프로그래밍

- 매크로 (#define)
  - 컴파일러가 소스코드를 컴파일하기 전에 매크로를 치환하는 방식으로 동작한다.
  - 매크로는 일반적으로 상수값이나 간단한 함수를 정의하는 데 사용된다.
  - 매크로는 텍스트 치환이기 때문에 타입 검사가 이루어지지 않으며, 복잡한 매크로는 디버깅이 어려울 수 있다.

- 매크로를 사용할 때의 유의사항
	- 코드가 단순 대체됨에 유의해야 한다.
	- 매크로문을 괄호로 감싸는 것이 안전하다.

	```cpp
	#define SQUARE(a) a*a
	result = 100/SQUARE(5); // 우리는 4를 기대하지만 100/5*5로 치환되어 결과는 100이 된다.

	#define SQUARE(a) ((a)*(a)) // 이렇게 괄호로 감싸면 100/SQUARE(5) 치환 시 100/((5)*(5))가 되어 올바른 결과인 4가 된다.
	```
