### Array

- 복합(compound) 데이터 타입 -> 클래스
- 동일한 타입인 요소들의 집합

- 개별 요소들에 직접적인 접근이 가능

- 사용 목적
  - 아래와 같은 코드를 효율적으로 바꿀 수 있다.
  ```
  int score_1 = 0;
  int score_2 = 0;
  int score_3 = 0;
  int score_4 = 0;
  ...
  int score_100 = 0;
  ```

### 배열의 특징

- 고정된(정해진) 길이
- 연속된 메모리 주소에 저장
- 인덱스를 통해 접근 가능
- 인덱스는 0부터 시작하며, 마지막 인덱스는 size-1
- out of bound 체크하지 않음
- 초기화 필요
- 효율적인 데이터 구조

-> 기본 배열은 잘 사용하지 않는다.
-> 그래도 공부하는 이유는 남이 짠 코드를 이해하기 위해, 추가된 기능들도 이에 기반해서 개발되기 때문

### 배열의 정의

- 기본 형
  Type array_name [size number]; -> 배열의 길이는 정해져있기 때문에 size number는 상수여야 한다.
  int socres[6];

  const int year = 365
  int number[year];

### 배열의 초기화

- int scores[5] = {10,85,21,34,56}
- int high_scores[10] = {3,5} // {3,5,0,0,0,....,0}

- int scores = {100,23,52} //자동으로 사이즈 측정
